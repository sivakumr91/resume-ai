How to Use the Resume Analyzer

1

Configure your environment (.env)

Open the .env file in the Source_Code directory and paste your Gemini API Key in the field: GEMINI_API_KEY=your_key_here.

Where to get a key? Go to Google AI Studio, log in with your Google account, and click Create API Key.

2

Start the Server

Run the local server using Python. In the terminal inside Source_Code/, run: python server.py. This starts the web application on http://localhost:3000.

3

Upload Resume & Job Description

Drop your resume in PDF/DOCX format, or paste the raw text. Then paste the job description you are targeting in the respective input areas.

4

Generate & Download the Analysis

Click "Analyze Resume" to invoke the Gemini API request. Review the generated match scores, skill gap analysis, contact integrity validations, and recommendations. Click "Export Report (PDF)" to save a clean printer-formatted document.

Fallback Parsing Engine

If the server is started without a GEMINI_API_KEY, the application will automatically enter Fallback Mode. It uses a local client-side NLP dictionary match. While this is less conversational than Gemini, it computes skill overlap, checks contact details, performs section checks, and runs entirely offline without external API dependencies!
